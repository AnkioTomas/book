# 更新日志

## [1.0.5] - 2026-06-27

### 新功能

- **AI 智能识别**：接入 OpenRouter / ChatGPT，自动检索豆瓣补全书籍元数据（书名、作者、简介、封面、评分、分类、标签），支持 SSE 实时推送进度
- **AI 批量识别**：选中多本书提交后台任务，AI 逐本检索并直接写库
- **AI 智能分类**：批量让 AI 根据书籍信息自动判断分类和标签，保留已有「已读」状态
- **AI 图书助手 Agent**：基于 Tool Calling 架构，具备本地藏书检索 + 豆瓣搜索 + 网页抓取能力
- **统计面板**：藏书概览（总量/已读/在读/平均评分/从未翻开）、分类分布、评分分布、近 12 月入库趋势、最近添加/最近阅读/从未翻开清单
- **增量同步任务**：全新书库增量同步引擎（SyncTask），支持远端文件列表驱动的删除传播、元数据 last-write-wins 仲裁、进度双向 mtime 仲裁、回写失败水位线保护
- **每小时自动同步**：注册定时任务，每小时自动执行一次 WebDAV 增量同步
- **封面刮削后台任务**：选中书籍批量提交后台封面刮削，从电子书文件中提取封面并上传 WebDAV
- **豆瓣图书搜索服务**：独立的 DoubanSearch 工具类，支持搜索、详情抓取、标签/简介/出版信息解析

### 改进

- 重构书籍模块的阅读进度和分类功能，字段语义纠正（favorite=分类、category=标签）
- 重构豆瓣搜索工具类，收敛搜索逻辑到统一入口
- 重构定时任务管理器和日志系统
- 重构仪表盘阅读器功能
- 优化侧边导航菜单折叠逻辑与路径处理
- 重构书籍和仪表盘界面样式与交互逻辑
- 重构配置模板文件结构，新增 `example.config.php`
- 删除过时配置文件并更新 .gitignore
- 更新 README 文档，补充 AI 功能、安装向导、统计面板等内容

### 修复

- 修复书籍同步时增量水位线控制问题

## [1.0.0] - 2026-05-07

### 新功能

- **书籍管理**：新增书籍管理功能与文件上传模块
- **WebDAV 同步**：实现 WebDAV 双向同步功能并支持系列信息处理
- **豆瓣集成**：重构书籍管理界面并集成豆瓣搜索功能
- **在线阅读**：将书籍管理页面从表格视图重构为卡片视图，并集成 Foliate.js 在线阅读功能
- **安装向导**：添加 Web 安装向导功能，首次部署通过浏览器填表完成
- **Calibre 集成**：新增电子书能力微服务（ebook-service），封装 Calibre CLI 为 HTTP API
- **Calibre 配置页面**：新增 Calibre 微服务配置页面，支持地址设置与连接测试
- **批量操作**：批量编辑、批量删除、批量刮削封面
- **删除重复**：新增删除重复书籍功能（按书名+作者判断，保留最早导入）
- **文件上传**：支持多文件上传、拖拽上传、大文件分片
- **阅读进度**：阅读进度双向同步，支持多种进度串格式解析
- **已读筛选**：添加书籍已读完筛选与进度同步功能
- **系列管理**：为书籍模型添加系列字段和数据库升级支持
- **封面代理**：WebDAV 封面代理与豆瓣封面下载
- **后台管理框架**：新增 WebDAV 配置界面与后台管理框架

### 改进

- 重构书籍管理界面，引入 DataTable 表格布局与拖拽上传功能
- 重构文件上传逻辑，支持批量处理与进度反馈
- 重构阅读进度模型和控制器
- 重构 BookDao，添加批量查询并优化删除逻辑
- 重构 Calibre 封装层，统一错误处理并简化 API 端点
